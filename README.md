# drawevolve.com

Marketing site for DrawEvolve — an iOS drawing studio with an AI coach that
remembers its past critiques. Dark "Night Studio" design: warm near-black,
bone text, sanguine accents, with session-record "receipts" (pass chips,
skill radar) in the memory and Evolution sections.

## Tech stack

- **Framework:** Next.js 14 (App Router), all server components except `FAQ`
  (accordion state) and `Reveal` (IntersectionObserver scroll reveals)
- **Styling:** Tailwind CSS + design tokens in `app/globals.css`
  (`--studio`, `--bone`, `--sang`, `--seam`, …)
- **Fonts:** Fraunces (display serif), Inter (body), JetBrains Mono (chips,
  timestamps) via `next/font`
- **Hosting:** Vercel, deploys from `main` — **pushing to main is publishing**

## Key routes

- `/` — the landing page (`components/marketing/*`)
- `/privacy`, `/terms` — required by App Store review
- `/.well-known/apple-app-site-association` — AASA for the iOS app's
  magic-link auth. **Do not remove.**
- `/sitemap.xml`, `/robots.txt` — generated by `app/sitemap.ts` / `app/robots.ts`

## Placeholders that need real content

- `lib/links.ts` — `TESTFLIGHT_URL` is still `REPLACE_ME`. Swap in the public
  TestFlight link (App Store Connect → TestFlight → Public Link), or the App
  Store URL once approved.
- `components/marketing/PortraitSketch.tsx` — placeholder SVG sketches used in
  the hero and "second pass" sections. Replace with real app screenshots /
  exported drawings (keep the pass-1 / pass-2 pairing).
- `lib/analytics.ts` — console no-op until an analytics key is configured.
